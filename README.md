# Навчальні завдання для Python Backend Розробника

## Про проект

Це **структурований навчальний курс** для підготовки junior python backend розробників. Курс складається з практичних завдань, які поступово вводять в основні концепції професійної backend розробки.

### Що входить в курс

1. **Завдання 1**: Базове налаштування середовища (Python + FastAPI + Docker)
2. **Завдання 2**: Доменна модель + Repository + інтеграція з FastAPI
3. **Завдання 3**: Персистентність даних з SQLite
4. **Завдання 4**: Консольний інтерфейс (CLI) для управління даними

### Навчальна мета

Курс навчає **архітектурним принципам** та **best practices** сучасної backend розробки:
- Розділення відповідальностей (Domain, Repository, Services)
- Тестування бізнес-логіки
- Робота з базами даних
- Створення API та CLI інтерфейсів
- Контейнеризація та deployment

## Доменна область

Проект використовує **модель системи бронювання поїздок**, що включає:
- Створення та управління поїздками
- Пошук по напрямках
- Бронювання місць
- Статистику доступності

Це дозволяє практикувати реальні сценарії, які зустрічаються в продакшені.

## Архітектурні принципи

Курс демонструє **чистоту архітектури**:
- **Domain Layer**: Бізнес-логіка незалежна від інфраструктури
- **Repository Pattern**: Абстракція доступу до даних
- **Service Layer**: Координація операцій
- **API/CLI**: Різні інтерфейси для однієї логіки

## Як проходити курс

1. **Починайте з Task-1** - базове налаштування
2. **Робіть завдання послідовно** - кожне будується на попередньому
3. **Використовуйте AI як помічника**, але розумійте код
4. **Документуйте промпти** в `prompts.txt` для кожного завдання
5. **Тестуйте** свій код на різних сценаріях


## Технічний стек

- **Python 3.11+**
- **FastAPI** - веб-фреймворк
- **SQLite**/**Postgres** - бази даних
- **Docker & Docker Compose** - контейнеризація
- **dataclasses** - доменні моделі
- **argparse** - CLI інтерфейс


## Набуті навички

Після проходження курсу ви зможете:
- Створювати структуровані Python додатки
- Розробляти REST API з FastAPI
- Працювати з базами даних через Repository паттерн
- Контейнеризувати додатки з Docker
- Створювати CLI інструменти
- Писати тести для бізнес-логіки
- Розуміти концепції Clean Architecture
