### Завдання №1. Базовий бекенд-сетап (Python + FastAPI + Docker)

**Мета:** підготувати робоче середовище для подальших задач.

#### 1. Локальне середовище

1. Встановити `python` (3.11+).
2. Створити окрему папку проекту `fastapi-project`.
3. Створити та активувати `venv` у цій папці.
4. Встановити залежності: `fastapi`, `uvicorn[standard]` (з використанням файлу requirements.txt)

Результат:

* Текстовий файл `README.md` з командами, які ти використовував для:

  * створення venv,
  * активації venv,
  * встановлення залежностей.


#### 2. FastAPI-сервер

1. Створити файл `main.py`.

2. Налаштувати FastAPI-додаток з одним GET-ендпоінтом:

   * `GET /health` → повертає JSON: `{"status": "ok"}`

3. Запустити додаток локально через `uvicorn` (наприклад, `uvicorn main:app --reload`).

4. Перевірити в браузері або через `curl`/Postman, що `http://localhost:8000/health` повертає потрібну відповідь.

Результат:

* Працюючий сервер.
* Скриншот або текстовий приклад відповіді `GET /health` додати в `README.md`.


#### 3. Docker + Docker Compose

1. Створити `Dockerfile`, який:

   * базується на офіційному Python-образі (наприклад, `python:3.11-slim`),
   * копіює код проекту,
   * встановлює залежності,
   * запускає `uvicorn main:app --host 0.0.0.0 --port 8000`.

2. Створити `docker-compose.yml`, який:

   * піднімає сервіс `api`,
   * форвардить порт `8000:8000`.

3. Запустити проект командою `docker-compose up` і перевірити:

   * що `http://localhost:8000/health` працює з контейнера так само, як і локально.

Результат:

* Працюючий контейнеризований FastAPI-сервер.
* У `README.md` додати команди для:

  * побудови образу (якщо потрібно),
  * запуску через `docker-compose`,
  * зупинки контейнерів.

### Що здати

1. Репозиторій з кодом проекту (`main.py`, `Dockerfile`, `docker-compose.yml`, `README.md`, `requirements.txt` або `pyproject.toml`).
2. Короткий опис у `README.md`, як запустити проект:
   * локально через venv;
   * через Docker Compose.
3. Якщо використовуєш AI для допомоги, створи ще один файл: prompts.txt, де записуй промпти і став їм оцінку на скільки вони були помічними для тебе. Наприклад:
```
1. prompt: що таке FastAPI?
score: 9/10
comment: AI дав хороше пояснення про FastAPI, хочу ще зрозуміти про ...
```
